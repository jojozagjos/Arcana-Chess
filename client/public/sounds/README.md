# Sound Files

Please place your sound files in this directory with the following names:

- `move.mp3` - Piece movement sound
- `capture.mp3` - Piece capture sound
- `ascension.mp3` - Game ascension sound
- `card-draw.mp3` - Arcana card draw sound
- `card-use.mp3` - Arcana card usage sound
- `check.mp3` - Check notification sound
- `victory.mp3` - Victory/game end sound

The sounds from `arcana-chess-placeholder-sfx` folder can be copied here and renamed accordingly.
