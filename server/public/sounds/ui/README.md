UI sound assets directory.

Recommended structure:
- /sounds/ui/move.mp3
- /sounds/ui/capture.mp3
- /sounds/ui/ascension.mp3
- /sounds/ui/card-draw.mp3
- /sounds/ui/card-use.mp3
- /sounds/ui/check.mp3
- /sounds/ui/victory.mp3

These are preloaded by the client sound manager under simple keys like `move`, `capture`, etc.
